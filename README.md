# GBI

Aplicación pensada para el aprendizaje de la gramática básica en inglés, nivel 3º-4º de primaria.
Se documentará aquí el funcionamiento y aplicación del aplicativo.



Notas:



\*\*\* Ubicación de oraciones según su lección \*\*\*



* wh\_questions --> ID 1-10, ID 31-33, ID 81-90
* pronouns\_to\_be --> ID 11-15, ID 21-22, ID 25-26, ID 41-50, ID 91-94, ID 101
* verb\_to\_do --> ID 16-20, ID 29-30, ID 61-70
* negations\_simple --> ID 23-24, ID 38-40, ID 66-67, ID 91-100
* pronouns\_have\_got --> ID 34-37, ID 51-60
* yes\_no\_questions --> ID 25-28, ID 36, ID 49-50, ID 58-60, ID 68-80



A tener en cuenta:



* ID 25-26 aparecen en dos lecciones porque tienen ambas: "yes\_no\_questions" y "pronouns\_to\_be"
* ID 36 aparece en dos lecciones: pronouns\_have\_got y yes\_no\_questions
* ID 49-50 aparecen en dos lecciones: pronouns\_to\_be y yes\_no\_questions
* ID 58-60 aparecen en dos lecciones: pronouns\_have\_got y yes\_no\_questions
* ID 91-94 aparecen en dos lecciones: pronouns\_to\_be y negations\_simple
